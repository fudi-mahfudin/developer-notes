# Case Study: Feature Delivery

Owner: Mahfudin  
Update cadence: Mingguan (saat latihan interview)

## Judul

Pengiriman fitur printing label farmasi berbasis QZ Tray di Siloam

## Situation

Tim farmasi membutuhkan alur cetak label yang cepat dan stabil untuk kebutuhan operasional harian. Proses lama memperlambat kerja dan rawan error.

## Task

Mendesain dan mengirim fitur end-to-end dari UI template editor sampai proses direct print yang terintegrasi ke workflow operasional.

## Action

- Merancang alur template editor agar operator bisa menyesuaikan label tanpa request engineering berulang.
- Membangun integrasi direct print berbasis QZ Tray.
- Menyesuaikan alur data agar input valid sebelum dikirim ke printer.
- Koordinasi dengan user operasional untuk iterasi cepat.

## Result

- Proses cetak menjadi lebih cepat dan konsisten.
- Mengurangi friksi operasional pada workflow labeling.
- Memberi bukti ownership lintas UI, integrasi, dan kebutuhan bisnis.

## Trade-off

- Memilih rollout bertahap per workflow untuk menjaga stabilitas produksi.
- Mengutamakan reliabilitas proses cetak dibanding menambah fitur non-kritis.

## Metrik yang Bisa Disebut di Interview

- Waktu rata-rata proses cetak sebelum/sesudah.
- Jumlah insiden terkait label printing sebelum/sesudah.
- Lead time perubahan template.

## Jawaban 90 Detik (Ringkas)

Saya memimpin delivery fitur printing label farmasi dari UI sampai integrasi printer menggunakan QZ Tray. Fokus saya adalah memastikan alur ini stabil untuk kebutuhan operasional harian, bukan hanya selesai secara teknis. Saya lakukan iterasi cepat dengan user operasional, memperketat validasi input, lalu rollout bertahap untuk meminimalkan risiko. Hasilnya proses cetak lebih konsisten dan friksi kerja tim menurun.
