# Persiapan Senior Full Stack Healthcare

Owner: Mahfudin  
Update cadence: Mingguan  
Target utama: Senior Full Stack Developer (frontend-heavy) di industri healthcare Indonesia

## Cara Pakai

- Mulai setiap hari Senin dari `03-weekly-sprint-planner.md`.
- Update status pipeline kerja setiap hari di `04-company-pipeline-tracker.md`.
- Catat latihan interview, coding, dan kendala di hari yang sama.
- Tinjau roadmap bulanan pada `02-6-month-execution-roadmap.md`.

## Siklus Eksekusi Mingguan

1. Tetapkan target mingguan (`03-weekly-sprint-planner.md`).
2. Kerjakan increment project portofolio (`05-portfolio-project-spec.md`).
3. Jalankan apply dan networking (`04-company-pipeline-tracker.md`, `11-linkedin-outreach-script-pack.md`).
4. Latihan interview dan system design (`09-interview-answer-bank.md`, `14-system-design-notes.md`, `15-coding-practice-log.md`).
5. Update metrik dan blocker di semua dokumen aktif.

## Metrik Keberhasilan

- 40-60 lamaran terarah per minggu.
- 3-5 respons recruiter/HR per minggu.
- 1-2 technical interview per minggu mulai bulan ke-3.
- 1 rilis portofolio production-like per bulan.
- 3 case study siap interview selesai maksimal bulan ke-2.
