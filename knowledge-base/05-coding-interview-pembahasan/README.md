## Cara Pakai Cepat

- Pilih 3-5 soal per hari (campur easy-medium-hard).
- Selesaikan dulu **tanpa** membuka file jawaban ideal 20–40 menit.
- Setelah itu, bandingkan dengan catatan di file `NNN-*.md` (kompleksitas, edge case, red flags).
- Wajib sebutkan time/space complexity saat latihan lisan.
- Tutup dengan 2 edge case per soal.
