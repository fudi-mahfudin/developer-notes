# Glossary Istilah Penting

- **Event Loop**: mekanisme yang mengatur eksekusi tugas async di JavaScript.
- **Call Stack**: antrean eksekusi fungsi aktif.
- **Promise**: representasi hasil operasi async di masa depan.
- **Concurrency**: menangani banyak tugas pada waktu yang tumpang tindih.
- **Parallelism**: menjalankan beberapa tugas benar-benar bersamaan.
- **Worker Thread**: thread tambahan di Node.js untuk tugas CPU-bound.
- **Idempotency**: eksekusi berulang menghasilkan efek akhir yang sama.
- **Retry**: mencoba ulang operasi gagal.
- **Timeout**: batas waktu operasi.
- **Circuit Breaker**: pola untuk menghentikan request sementara saat dependency gagal terus-menerus.
- **Race Condition**: hasil bergantung urutan eksekusi yang tidak terkontrol.
- **Deadlock**: dua proses saling menunggu resource tanpa selesai.
- **Backpressure**: mekanisme menahan laju producer agar consumer tidak overload.
- **SLA**: komitmen tingkat layanan.
- **SLO**: target internal tingkat layanan.
