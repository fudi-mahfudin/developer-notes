## 12 - Exit Criteria per Level
- **Junior**: memahami modularity, layering, dan basic design pattern
- **Mid**: mampu mendesain service/module boundary dan memilih pattern yang tepat
- **Senior**: mampu menentukan arsitektur, trade-off, reliability, dan governance teknis lintas sistem
